# 🛠️ Estofaria Qualidade - Landing Page

Projeto de uma landing page moderna e responsiva desenvolvida para uma estofaria, com foco em apresentação de serviços e conversão de clientes via WhatsApp.

---

## 🚀 Tecnologias Utilizadas

- **Next.js** — Framework React para renderização híbrida (SSR/SSG)
- **React** — Biblioteca para construção da interface
- **TypeScript** — Tipagem estática para maior segurança e escalabilidade
- **CSS Modules / Tailwind (ajuste conforme usou)** — Estilização da aplicação
- **Git & GitHub** — Versionamento e controle de código

---

## 📱 Funcionalidades

- Layout moderno e responsivo (mobile, tablet e desktop)
- Menu adaptativo com versão mobile (hambúrguer)
- Seção de serviços
- Seção de contato
- Integração com WhatsApp para envio de mensagens
- Validação de formulário
- Máscara para telefone
- Navegação fluida entre seções
- Deploy contínuo (CI/CD via Vercel)
